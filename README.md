# kpi-kb
El plugin KPI surge de la necesidad de acoplar a datal un módulo capaz de crear, editar y publicar métricas.

A través de él es posible visualizar comparaciones de datos como resultados esperados vs. resultados reales, resultados obtenidos entre niños vs. adultos, cumplimiento de objetivos y una infinidad de posibilidades más, de acuerdo a la necesidad puntual de cada municipio.
